## 📡 Architecture Overview

This project demonstrates a scalable, globally distributed live video streaming system for baseball games using AWS services such as MediaLive, MediaPackage, CloudFront, Lambda, and DynamoDB.

The system is designed for high availability, low latency, and global content delivery.
